About Me

The first time I clicked compile and run on a "Hello World!" program in Java and watched the words appear on the screen was when fell in love with coding. Ever since then, the idea that you can use a computer and put in a few lines of code to get a desired output is something that has constantly amazed and inspired me.


I carried that passion into taking up Computer Science as an undergraduate and combine it with my love for art and creativity to delve into web development. Now, I am passionate about building seamless, user-friendly web experiences.
I am currently learning web design and how to apply it into projects that I build.
When I'm not coding/designing, I can be found reading or sketching.
